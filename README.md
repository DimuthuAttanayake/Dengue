# Dengue
An auto updating map for weekly dengue cases in Sri Lanka. Dengue is the mosquito borne disease that could be deadly. 
This map uses data from [Ministry of Health, Sri Lanka](https://www.epid.gov.lk/weekly-epidemiological-report/weekly-epidemiological-report).

This is the final project for [Computing Foundations], Fall 2025.
